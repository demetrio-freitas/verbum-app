# VERBUM — Catecismo da Igreja Católica

> Feature Spec · Marketplace + Estudo Guiado
> 7BIS Startup Studio · Abril 2026

---

## 1. Visão Geral

O Catecismo da Igreja Católica (CIC) integrado ao Verbum como ferramenta de estudo, referência e aprofundamento doutrinário. Diferente da Bíblia (que é leitura devocional diária), o Catecismo é referência consultiva — o fiel busca quando tem uma dúvida, quando quer estudar um tema, ou quando segue um plano de formação.

**Modelo de monetização:** híbrido.

```
GRATUITO:
├── Busca por tema/palavra (resposta com parágrafo exato)
├── Link automático do evangelho do dia para o Catecismo
├── Índice navegável (4 partes, 2.865 parágrafos)
└── 1 plano de estudo gratuito ("7 dias de introdução")

CATECISMO COMPLETO (R$19,90/ano — compra no marketplace):
├── Texto integral navegável offline
├── Destaques e anotações sincronizadas
├── Busca full-text avançada
└── Acesso a todos os planos de estudo

PREMIUM VERBUM (R$49,90/ano — inclui tudo):
├── Catecismo completo incluso
├── + Bíblia features premium
├── + Stories exclusivos
├── + Áudio narrado
├── + Tudo do Premium
└── Melhor custo-benefício
```
